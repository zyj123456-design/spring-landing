# Spring Festival Landing Page

A beautiful, interactive Spring Festival landing page with real-time greeting wall built with Next.js and deployed on Zeabur.

## Features

- 🎊 Interactive New Year greetings wall
- 🎨 Beautiful Spring Festival theme
- 📱 Mobile-responsive design
- 🔒 Secure content filtering
- 🚀 Fast performance with Next.js 15

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Deployment**: Zeabur

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## API Routes

- `GET /api/greetings` - Get all greetings
- `POST /api/greetings` - Create new greeting
